此專案為 https://github.com/derekjan1240/webRTC-test 206 前端測試使用

### `開啟 206 後端 server`

前往 https://github.com/derekjan1240/webRTC-test clone 並開啟 Server

### `yarn install`

請先將需要的套件安裝完成

### `yarn run start`

開啟 Server 前往 [http://localhost:3000](http://localhost:3000)
